# Azure Zero Trust Security Lab

## 🔐 Overview
This project simulates a cloud security environment using Azure, focusing on identity-based security, Zero Trust principles, and threat detection.

## 🎯 Objective
Demonstrate how to protect cloud environments using:
- Identity and Access Management (Entra ID)
- Threat detection (Microsoft Sentinel)
- Zero Trust concepts

## 🧱 Architecture
![Architecture](architecture/diagram.png)

## 🔍 Detection Scenario
Simulated brute-force attack detection using KQL queries.

## ⚙️ Technologies
- Azure
- Microsoft Sentinel
- Entra ID
- Defender for Cloud

## 💡 Key Learnings
- Identity is the new perimeter
- Monitoring is critical
- Zero Trust reduces attack surface
